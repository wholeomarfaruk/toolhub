# Laravel Starter Kit 🚀

A production-ready Laravel 12 SaaS starter template with multi-panel authentication, role & permission management, admin dashboard, and modern frontend tooling.

🔗 Repository: https://github.com/wholeomarfaruk/laravel-starter-kit.git

---

## ✨ Features

- ✅ Laravel 12 (Latest)
- ✅ Livewire v4
- ✅ Multi-Panel Architecture (Admin + User)
- ✅ Separate Admin Authentication
- ✅ Role & Permission System
- ✅ Jetstream Authentication
- ✅ Vite + TailwindCSS 4
- ✅ AlpineJS Integration
- ✅ File Upload System (FilePond)
- ✅ Modern UI Plugins Included
- ✅ Ready-to-use SaaS Foundation
